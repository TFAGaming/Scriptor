# Scriptor

Scriptor is a powerful and open-source IDE based on Java Swing. It provides many features, fully customizable, and beginner-friendly!

- Syntax highlighting
- Autocomplete*
- File system tree
- Built-in terminal
- Plugins and themes
- SQLite and Markdown viewer

*: Still under development.
